google, apple, kakao 등의 인증을 처리할 oauth2 를 구현한다.

enum OAuth2Provider
{
    Google,
    Apple,
    Kakao
}

route 에선 {provider}/login, {provider}/callback 를 구현한다.

SocialLoginUsecase trait 으로 처리
- login : redirect url 반환
- callback : code 를 받아서 token 을 받고, user 를 반환

GoogleLoginUsecase 만 구현한다.
- callback 에서 사용자가 없을시 자동 회원가입 처리한다.

GoogleLoginUsecaseConfig
{
    client_id: String,
    client_secret: String,
    redirect_uri: String,
    scope: String
}

http 연결은 reqwest 를 사용한다.

사용자 식별용도이므로 code 로 token 을 추출하여 id_token 의 sub 값을 식별자로 사용한다.

google auth url : "https://accounts.google.com/o/oauth2/v2/auth?response_type=code\
            &client_id={}&redirect_uri={}&scope={}&state={}&access_type=offline&prompt=consent"

google token url : "https://oauth2.googleapis.com/token"
